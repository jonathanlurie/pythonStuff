ThousansWord is an app to extact the 1000 most used word from a text.
This is supposed to provide a vocabulary starter kit for learning a foreign language.

The input text must be relevant:
- not one-genre story (i.e. only scifi)
- recent
- equivalent to a book of hundreds of pages


ebook (epub) can be converted to plain text with:

http://www.epubconverter.com/epub-to-txt-converter/
